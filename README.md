### Setup

```shell
npm install ngx-mqtt-lite --save
```

### Configuration

Add scripts in `angular.json`

```json
[
  {
    "scripts": [
      "./node_modules/mqtt/dist/mqtt.min.js"
    ]
  }
]
```
